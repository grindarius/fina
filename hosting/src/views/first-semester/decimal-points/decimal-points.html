Decimal Points

Finding how many points we need is hard.

Description

It is a point or dot we use to separate the whole number part from the fractional part of a number. A decimal number is a number that consists of a whole number and a fractional part. The decimal point separates the whole number from the fractional part.

Examples

Take a look at this number:

258.16

You can see that the number is separated by a dot. We call the part before the dot 'Whole number', and we call the part after the dot 'Fractional part'.

We start counting the amount of decimal points by counting the amount of number after the dot. So, from the example, this would have 2 decimal points.
