Significant Digits

Like how significant it could be?!?

Description

Significant Digits, or is the number that is absolutely necessary to indicate the quantity of something. If a number expressing the result of measurement of something (e.g., length, pressure, volume, or mass) has more digits than the digits allowed by the measurement resolution, only the digits allowed by the measurement resolution are reliable so only these can be significant figures.

Rules

  1. All non-zero numbers are significant digits, e.g., 846 has 3 significant digits, 8, 4, and 6.

  2. All zeros between non-zero numbers are significant digits, e.g., 5005 has 4 significant digits, 5, 0 (on hundreds place), 0 (on tens place), and 5.

  3. Leading Zeros, meaning zeros before any non-zero numbers, is never significant in both float and integer, for example.

    • 0.0074 has 2 significant digits, 7 and 4.

    • 013 has 2 significant digits, 1 and 3.

  4. All zeros after non-zero numbers (called Trailing Zeros), well, this depends.

    • If it's an integer e.g. 400, then we won't count those 2 zeros, so this one will have 1 significant digits, 4.

    • If it's a float without zeros behind the dot, e.g., 540., then we count the zero before the dot too, so this one will have 3 significant digits, 5, 4, and 0.

    • If it's an ordinary float, e.g. 400.00, then we count all the zeros both before and after the dot, so this one will have 5 significant digits, 4, 0 (on tens place), 0 (one ones place), 0 (on tenths place), and 0 (on hundredths place).

  5. Numbers in scientific notation , all digits comprising are significant by the rules above, and are NOT significant, e.g., has 3 significant digits, 5, 0, and 2.

Rounding a number to a desired Significant Digits

In previous session, you have learned about finding significant digits. In this part, we will teach you how to round a number to a specified significant digits.

Rules

  1. Look at the first non-zero digit if rounding to one significant digit.

  2. Look at the digit after the first non-zero digit if rounding to two significant digits.

  3. Draw a vertical line after the place value digit that is required.

  4. Look at the next digit.

    • If it's 5 or more, increase the previous digit by one.

    • If it's 4 or less, keep the previous digit the same.

  5. Fill any spaces to the right of the line with zeros. stopping at the decimal point if there is one.

  6. If the significant digits you want to round to is smaller than the number (50,000 round to 3 SD), you can round them up into Scientific Notation () form too.
