What is FINA

FINA (pronounced FIN-NA), short for Fabulous and Interactive Numerical Analysis Calculator, is a tool to learn Numerical Analysis. Since I found so many struggles learning this subject. So I decided to make this a Web App for people who wanted to study it.

Features

  • Calculate answers for many topics in Numerical Analysis.

  • Short and straight to the point explanation and examples.

  • Easy to use UI

  • You can send a link to your friend to show them the answer. Just copy the link that ends with /calculate with all the querystring to your friend and they can see the result immediately.

Caveats

This website is far from being done, and there are some points I wanted to tell you guys before you start taking a look on it.

  • The graph component that's used to draw the graph can display false data when your function is not continuous, meaning it has some gaps in the function, or some values results in Infinity or -Infinity. If those things are found in the function. our grapher component will change the infected value to 0.

  • There is a settings in the code where when is not any type of finite number, it will break the calculation loop. That condition might result in sometimes the API would send nothing back from the calculation.

  • Stuffs for second semester are in development. But all PRs welcome though if you wanted to be a part of us.

  • Calling API straight without the server could give you a funky error.

  • The errors aren't pretty.
